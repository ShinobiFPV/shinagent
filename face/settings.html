Pairing (8BitDo Zero 2)
Use GAMEPAD mode, not keyboard mode:
1. Clear old pairing: bluetoothctl remove <MAC>
2. Power on: hold B + START (LED blinks once per cycle)
3. Pair mode: hold SELECT 3 sec (LED rapid-blinks)
4. Pair via bluetoothctl (scan on → pair → connect → trust)
5. Recommended: press LEFT + SELECT on the controller for hat-mode D-pad

⚠ Avoid R + START (keyboard mode) -- causes typing conflicts in the web app
